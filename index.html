About This Project

WM‑Engineering is a collection of structural engineering analysis tools developed over 45 years of professional consulting practice. These applications support wood and steel beam design, seismic moment frame analysis, shear wall evaluation, and detailed load calculations following AISC, ASCE 7, CBC, and NDS standards.

Main Applications

  • Steel Moment Frame App — AISC seismic design, RBS logic, deflection checks
  • Wood Beam Design App — NDS‑based beam sizing with Kivy GUI
  • Beam Analysis — reactions, shear/moment diagrams, zero‑shear detection
  • Beam Calculator (Graph) — visual diagrams with Matplotlib
  • Beam Calculator (No Graph) — lightweight calculation tool

Windows Installation

  • Install Python 3.8+ (check “Add Python to PATH”)
  • Verify with python --version
  • Install dependencies: pip install -r requirements.txt
  • Run apps from the apps/ folder
